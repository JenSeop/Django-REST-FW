# Django REST FW
Django, Rest Framework 학습을 위해 책관리 API를 제작한 레파지토리입니다.<br>
Rest Framework의 기술 중 FBV, CBV, Mixins, generics, Viewset, Router의 학습에 주안을 뒀습니다.
